  $ lsof -i :5433
  $ kill -9 <PID>
  Or edit docker-compose.yml to use port 5434

Solution:
  $ docker compose down -v
  $ docker compose up -d
  Wait 2 minutes

Solution: Generate new token in UI, use correct token in command

Solution: Make sure both containers are running: docker compose ps
